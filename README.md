## Padrões de Projeto em Java (Cod3r)

Um repositório com implementações dos principais padrões de projeto (GoF), incluindo criacionais, estruturais e comportamentais, com exemplos práticos em Java.

###🎯 Visão Geral
Este projeto surgiu após a conclusão do curso “Padrões de Projeto em Java” da Cod3r, onde explorei:

- Padrões GoF: Factory, Singleton, Observer, Strategy, Decorator, entre outros.
- Categorias: Criacionais, Estruturais, Comportamentais.
- Aplicações práticas: estrutura de código, melhor uso em situações reais.
